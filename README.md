# StarWarsCharacters
Kotlin app that shows a list of characters using the api SWAPI (https://swapi.dev)

This is a clean architecture app example built with

- Coroutines
- MVVM
- Extension Functions
- Retrofit
- Room
- Navigation Components
- DataBinding
